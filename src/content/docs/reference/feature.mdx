このページでは、やみすきーの技術的な設定と仕様を解説します。「プライバシーファースト」の理念がどのように実装されているかを確認できます。

## プライバシー保護

やみすきーでは、運営側であっても必要最小限の情報しか閲覧できないよう設計しています。また、ログインしていないユーザーには情報を公開しない方針です。

## やみモード<Original />

やみすきー独自の機能で、「病み」や「闘」を吐き出すための専用モードです。

### やみモードとは

やみすきーには2種類のノート（投稿）があります：

- **通常ノート**: 一般的なSNS投稿。すべてのタイムラインに表示される
- **やみノート<Original />**: 「病み」を吐き出すための投稿。やみTL<Original />専用で、通常のTLには流れない

やみモードをONにすると、やみノートの投稿・閲覧が可能になります。OFFの状態では自分のやみノートも含めて完全に非表示になるため、「今は見たくない」ときに切り替えられます。

### やみノートの特徴<Original />

| 特徴 | 通常ノート | やみノート |
|------|----------|----------|
| 表示場所 | 各種TL | やみTL専用 |
| リノート | 可能 | **不可** |
| 連合先 | 全サーバー | 信頼済みサーバーのみ |
| 閲覧条件 | なし | やみモードON |

### モード別の表示

#### やみモードOFF（通常モード）

メンタルヘルス保護のため、**やみノートは完全に非表示**になります。

| タイムライン | 通常ノート | やみノート |
|-------------|:----------:|:----------:|
| ホームTL | ✅ | ❌ |
| やみTL | − | ❌（空） |
| ローカルTL | ✅ | ❌ |
| ソーシャルTL | ✅ | ❌ |
| グローバルTL | ✅ | ❌ |
| ユーザープロフィール | ✅ | ❌ |

#### やみモード<Original />ON

やみノートが閲覧可能になります。**やみノートはやみTLに分離**され、他のTLには流れません。

| タイムライン | 通常ノート | やみノート |
|-------------|:----------:|:----------:|
| ホームTL | ✅ | ❌（分離） |
| **やみTL** | − | **✅** |
| ローカルTL | ✅ | ❌ |
| ソーシャルTL | ✅ | ❌ |
| グローバルTL | ✅ | ❌ |
| ユーザープロフィール | ✅ | ✅ |

### やみTLのフィルター設定<Original />

やみTL<Original />に表示するやみノートの範囲を設定できます：

| 設定 | 説明 |
|------|------|
| フォロー中ユーザーのやみノートを表示 | フォロー中ユーザーのやみノートを表示/非表示 |
| フォロー外ユーザーのパブリックやみノートを表示 | フォロー外のパブリックやみノートを表示/非表示 |

**自分のやみノートは常に表示**されます（設定の影響を受けません）。

設定の組み合わせ例：
- 両方ON: すべてのやみノートを表示
- 両方OFF: 自分のやみノートのみ表示（やみノート一覧として使える）

### DMの扱い

ダイレクトメッセージ（DM）は特別扱いです：
- やみモードON時、自分が宛先に含まれていれば常に表示
- フィルター設定の影響を受けません

## ロール

やみすきーの権限システムです。ベースロール + コンディショナルロール（自動付与）+ マニュアルロール（申請/付与）の3層構造になっています。

### ベースロール（全ユーザー共通）

すべてのユーザーに適用される基本設定です。セキュリティとプライバシー保護のため、**最小権限の原則**で設計されています。

<Aside type="note" title="ポイント">
- LTL/GTL/やみTL<Original />、パブリック投稿、検索などはデフォルト無効（コンディショナルロールで有効化）
- フォロー/フォローされる、アイコン更新、広告非表示などは有効
- 各種インポート、招待コード発行、フォロー解除/ブロック通知は無効
</Aside>

#### 制裁ロール（モデレーション用）

[モデレーション原則の対応の段階](/reference/moderation#対応の段階)における「制限」措置として使用されるロールです。違反内容に応じて単独または組み合わせて付与されます。

| ロール | 役割 | 対象となる違反 |
|--------|------|----------------|
| サーチバン | 検索を禁止 | ストーカー的行為 |
| フォローバン | フォローを禁止 | つきまとい |
| サイレンス | 発信を制限 | 対人トラブル |
