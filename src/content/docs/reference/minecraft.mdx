## Bedrock Edition

プレイ > サーバー > サーバーを追加

- 「サーバーアドレス」を `mmc.yami.ski`
- 「ポート」を `1373`

## Java Edition

マルチプレイをクリック

- 「バージョン」`1.21.1` をインストール
- 「サーバーアドレス」を `mc.yami.ski`

## サーバー構成（PaperMC + Geyser/Floodgate）

### 概要

- **サーバーソフトウェア**: PaperMC（高性能なSpigotフォーク）
- **クロスプラットフォーム対応**: Geyser + Floodgate
- **コンテナ化**: Docker + Docker Compose
- **管理ツール**: Portainer
- **ネットワーク**: PlayIt（トンネリング）- オプション
