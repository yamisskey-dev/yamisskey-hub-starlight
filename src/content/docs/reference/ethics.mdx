## はじめに

このページでは、やみすきーとYAMIエコシステムの根底にある倫理観と思想的背景を説明します。

具体的なマナーや過ごし方については「[唯のやみかた](/guides/manner/)」を、運営の実務については「[運営について](/guides/operation/)」を参照してください。

## 「インターネット3」としてのYAMI

YAMIエコシステムは、Web3（ブロックチェーン経済）とは異なる意味での「インターネット3」を目指しています。

現在のSNS（X/Twitter、Instagram等）は、承認地獄・数値競争・アルゴリズム支配に特徴づけられます。にゃるら氏の著作「インターネット2」は、過去の牧歌的なインターネット（2000年代の個人サイト、2ch等）への憧憬を描いていますが、それは一時的な避難であり、結局SNSに戻ってしまう循環構造があります。

YAMIは、その循環からの脱出を目指します：

- 数値競争・拡散・承認を目的にしない設計
- 弱者が壊れずにいられる居場所
- 一時的な憧憬ではなく、持続可能な回復空間
- 自覚的に構築された、新しいインターネットのあり方

### 巨大プラットフォームとの対比

| 巨大プラットフォーム | YAMIエコシステム |
|---------------------|-----------------|
| アルゴリズムによるフィード最適化 | 時系列表示、ユーザーが制御 |
| 行動データの収集・広告利用 | ノーログポリシー、広告なし |
| 中央集権的な運営 | 分散型プロトコル、コミュニティ運営 |
| 企業による非公開の意思決定 | DAOによる透明な投票 |
| 利用規約でAI学習を許諾 | データ収集を最小化 |

## メンタルファーストとプライバシーファースト

やみすきーの二つの基本理念です。

### メンタルファースト

つらい気持ちを言葉にすることを大切にしています。

**「病み」「闇」を受け入れる**

やみすきーという名前が示す通り、ここは「病み」や「闇」を言葉にできる場所です。

- 「死にたい」「消えたい」という気持ち
- 「助けて」「話を聞いて」という呼びかけ
- 「つらい」「しんどい」という弱音
- 日常では言えない負の感情、苦しみ、脆さ

他のSNSでは「うざい」「重い」「構ってちゃん」と言われるような表現も、ここでは否定されません。

法律と[利用規約](/reference/term/)、[モデレーション原則](/reference/moderation/)に違反しない限り、あなたの投稿を検閲しません。

**「見たくない」も正当な権利**

一方で、「病み」「闇」を表現する自由がある一方で、「見たくない」という自由もあります。

- 今日は人の苦しみを受け止める余裕がない
- 特定の話題がトリガーになる
- この人の投稿は自分にとってつらい

そんなときは、ブロックやミュートを使って距離を取ることができます。それは冷たいことでも、悪いことでもありません。自分を守るための正当な手段です。

### プライバシーファースト

個人情報の扱いに細心の注意を払います。

- [ノーログポリシー](/reference/privacy/)：IPアドレスを記録しません
- Tor/VPN接続を歓迎
- メールアドレス不要でアカウント作成可能
- 匿名性を大切にする

## 自律性とコミュニティ

やみすきーでは、**ユーザー自身が自律的に行動すること**を重視しています。

### 自分を守る

ブロックやミュートを積極的に使ってください。

- 今日は人の「やみ」を見たくない → やみモードOFF
- この人の投稿がつらい → ミュート
- この人と関わりたくない → ブロック

これは冷たいことでも、悪いことでもありません。自分を守るための正当な手段です。

### 自律的に行動する

運営に依存せず、自分で判断してください。

- ユーザー同士で解決できることは、解決する
- グレーゾーンは自分で判断する
- 運営に「どうすればいいですか？」と聞く前に、自分で考える

運営は魔法使いではありません。すべてのトラブルを解決できるわけではありません。

**「運営に依存しない」ことが、やみすきーを守ることにつながります。**

詳しくは「[運営について](/guides/operation/)」を参照してください。

## 表現と支え合いのバランス

やみすきーでは「自分を表現する自由」と「みんなで支え合うこと」のバランスが大切にされています。

- どんな気持ちを表現しても否定されない
- 他の人の気持ちを批判するのはNG
- 一人に頼りすぎない
- 相手にも生活があることを理解する
- 返信には時間がかかることもある

### 「みかた」の二重性

「味方」になりたいと思っても、人によって「見方」が違うことを理解する必要があります。

- 全員が全員の味方になれるわけではない
- 自分の心の健康が第一
- 相手の「見方」を否定せず、理解しようとする
- できる範囲で支え合うことの大切さを忘れない
- 時には「見守る」だけでも十分な「味方」になれる

完璧な人間関係なんてないし、みんながみんなと仲良くなれるわけではありません。それぞれの距離感や境界線を大切にしながら過ごせたらいいな、と思っています。

## 「病み方」について

やみすきーは「認められにくい人たちの居場所」を目指している場所です。他の人を傷つけない限り、どんな気持ちも受け入れたいと思っています。

完璧じゃなくていい。調子の良い日も悪い日も、あなたはここにいていいんです。

病むことそのものは悪いことじゃありません。病みながらも、病みつつも、どうやって自分らしく生きていくか—それが「やみかた」です。

辛い日も、寂しい日も、怒りで震える日も、ただ泣きたい日も。

たとえどんなに暗い気持ちでも、忘れないでほしい。

あなたには味方がいる。

「病んでるね」って否定じゃなくて、「病んでるね、私も」って共感し合える場所。それがやみすきーです。
