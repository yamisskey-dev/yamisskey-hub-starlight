## 前文

本モデレーション原則は、やみすきーにおけるモデレーションの執行手続きを定めるものです。

本原則は[利用規約](/reference/term/)の下位規範であり、利用規約に定める4つの倫理的原則（自律尊重、危害原則、明示性、平等適用）に基づきます。

---

## 第1章 禁止事項

以下の行為を禁止します。**本章に列挙されていない行為は禁止されません。**

### 第1条 自律侵害行為

1. **返信の強要**: 「返信しないと〜する」「無視するな」等、返信を強要する表現
2. **行動の束縛**: 「〇〇時まで起きていて」「他の人と話すな」等、他者の行動を束縛する表現
3. **接触拒否の無視**: 相手が明示的に距離を置く意思を示した後も、接触を続ける行為
4. **同調の強制**: 自己の感情表現への同調を他者に強制する行為

### 第2条 脅迫的行為

1. **脅迫的支援要求**: 「〇〇してくれないと死ぬ」「助けてくれないなら自傷する」等の表現
2. **罪悪感の付与**: 支援を断った相手を責める、または罪悪感を与える行為
3. **試し行為**: 「私のことを本当に心配しているなら〜してくれるはず」等、相手を試す表現
4. **脅迫・恐喝**: 「〇〇しないと〜する」「〜を公開するぞ」等の脅迫・恐喝
5. **危害の示唆**: 身体的または精神的危害を示唆する表現

### 第3条 権利侵害行為

1. **プライバシー侵害**: 他者の個人情報（本名、住所、電話番号、メールアドレス、職場、学校、病歴、治療歴、通院状況等）を本人の同意なく公開する行為
2. **名誉毀損**: 特定個人に対する、事実に基づかない中傷
3. **なりすまし**: 他者または運営者になりすまして投稿する行為
4. **ハラスメント**: 相手が明示的に拒否した後も執拗に接触を続ける行為、ストーキング行為、セクシュアルハラスメント

### 第4条 危害誘導行為

1. **自殺・自傷の推奨**: 自殺または自傷を推奨する表現
2. **具体的方法の提示**: 自殺または自傷の具体的方法を提示する行為
3. **犯罪の教唆**: 犯罪行為を教唆、幇助、または実行する行為

### 第5条 差別的行為

1. **差別的攻撃**: 人種、国籍、民族、性別、性的指向、性自認、宗教、障害等を理由とした攻撃
2. **排除の扇動**: 特定の属性を持つ集団全体に対する侮辱または排除の扇動

### 第6条 運営妨害行為

1. **虚偽の通報**: 虚偽の内容による通報を繰り返す行為
2. **システム攻撃**: DDoS/DoS攻撃、不正アクセス、サーバーへの過負荷等
3. **スパム**: 同一または類似の内容を短時間に大量投稿する行為（目安：1時間に10回以上）
4. **運営者への脅迫**: 運営者に対する脅迫または過度な要求

### 第7条 コンテンツ規制

1. **NSFW指定の不備**: 性的、暴力的、またはその他センシティブなコンテンツをNSFW指定なしで投稿する行為
2. **未成年者保護違反**: 未成年者が閲覧可能な状態で成人向けコンテンツを投稿する行為
3. **違法コンテンツ**: 児童の性的搾取、リベンジポルノ、犯罪記録等の違法コンテンツの投稿

## 第2章 許容事項

以下の行為は明示的に許容されます。**これらの行為を理由とする処分は禁止されます。**

### 第8条 許容される表現

1. **感情の表現**: 「死にたい」「辛い」「消えたい」等の感情表現
2. **助けを求める表現**: 「助けてほしい」「話を聞いてほしい」等の表現（強要を伴わない限り）
3. **批判的意見**: 運営、他ユーザー、またはコミュニティに対する批判的意見の表明（事実に基づく限り）
4. **意見の相違**: ユーザー間の意見の相違、議論、論争
5. **不快な表現**: 禁止事項に該当しない限り、他者が不快に感じる表現

### 第9条 許容される行動

1. **関与の拒否**: 他者の投稿への反応を拒否すること
2. **距離を置くこと**: ミュート、ブロック、フォロー解除等により距離を置くこと
3. **通報**: 禁止事項に該当すると思われる行為を通報すること
4. **異議申し立て**: 処分に対して異議を申し立てること

## 第3章 処分

### 第10条 処分の段階

禁止事項への違反が確認された場合、以下の段階に従って処分を行います。

1. **警告**: 初回の違反に対する警告。違反内容と該当条項を明示して通知。
2. **制限**: 警告後も改善が見られない場合の機能制限。期間を明示して通知。
3. **一時停止**: 深刻または繰り返される違反の場合のアカウント凍結。期間を明示して通知。
4. **永久停止**: 悪質な違反、または一時停止後も改善が見られない場合のアカウント永久凍結。

### 第11条 即時永久停止

以下の行為については、警告なしに即時永久停止とします。

1. 法令違反（犯罪行為、児童ポルノ、違法薬物取引等）
2. システム攻撃（DDoS攻撃、クラッキング、不正アクセス等）
3. 深刻なプライバシー侵害（個人情報の大規模な暴露等）
4. 重大な脅迫または恐喝
5. 永久停止後の別名義による再登録

### 第12条 処分の手続き

1. **証拠の保全**: 処分の根拠となる投稿またはログを記録する
2. **理由の明示**: 違反した条項を具体的に通知する
3. **合議**: 処分は複数のモデレーターによる合議を経て決定する
4. **記録**: すべての処分について、日時、理由、経緯を記録する
5. **異議申し立て**: ユーザーは処分から14日以内に異議を申し立てることができる

### 第13条 異議申し立て

1. 処分を受けたユーザーは、処分通知から14日以内に異議を申し立てることができる
2. 異議申し立ては、元の処分を決定したモデレーター以外が審査する
3. 審査結果は30日以内に通知する
4. 審査結果に不服がある場合、DAO投票による最終判断を求めることができる

## 第4章 運用体制

運営体制の基本構造は[利用規約 第5条](/reference/term/#第5条運営体制)に定めます。本章ではモデレーション固有の運用規則を定めます。

### 第14条 運営が行わないこと

1. ユーザー間の仲裁または調停
2. 予防的な監視または指導
3. 禁止事項に該当しない行為への介入
4. 「雰囲気」「空気」「コミュニティに合わない」等の曖昧な理由による介入

### 第15条 運営の立場

1. やみすきーは医療福祉機関ではない
2. 運営は医療的判断または福祉的支援を行わない
3. 専門的な支援が必要な場合は、医療機関または相談窓口への相談を推奨する

---

## 第5章 解釈規則

### 第16条 厳格解釈

1. 本原則は、その文言に従って厳格に解釈する
2. 類推解釈は禁止する
3. 拡大解釈は禁止する
4. 「等」の例示は、例示された行為と同等の具体性および明確性を持つ行為に限り適用する

### 第17条 疑義の処理

1. 禁止事項への該当に疑義がある場合、該当しないものとする（推定無罪）
2. 解釈に疑義が生じた場合、ユーザーの権利を保護する方向で解釈する
3. 新たな解釈が必要となった場合、その解釈を公開し、本原則を改定する

### 第18条 禁止される運営行為

運営は、以下の行為を行うことができない。

1. 本原則に明記されていない理由による処分
2. 投稿前の内容審査または予防的な投稿制限
3. 運営者の個人的な好悪に基づく介入
4. 禁止事項の類推適用または拡大解釈
5. 処分理由を明示しない処分

## 第6章 改定手続き

本原則の改定手続きは、[利用規約 第17条](/reference/term/#第17条利用規約の変更)に従います。

本原則は利用規約の下位規範であるため、利用規約の改定要件より緩和された手続きで改定できます：

1. 管理人が提案し、7日間の意見聴取期間を設ける
2. 反対意見が一定数を超えた場合、DAO投票に付する
3. 意見聴取期間終了後、施行する

ただし、本原則の改定が利用規約と矛盾する場合、利用規約が優先されます。

---

## 第7章 例外規定

### 第19条 緊急対応

以下の場合に限り、運営は本原則に定めのない対応を取ることができる。

1. 自殺または自傷の切迫した危機が認められる場合（専門機関への誘導、一時的な投稿制限）
2. 法令に基づく要請がある場合（捜査機関への協力、裁判所命令への対応）
3. サーバーの技術的存続に関わる緊急事態

緊急対応を行った場合、24時間以内に理由を公開する。
