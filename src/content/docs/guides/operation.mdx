やみすきーでは**ユーザー自身が自律的に行動すること**を重視しています。

## 運営がすること・しないこと

やみすきーの運営は、みんなが安全に過ごせるように見守っていますが、**すべてのトラブルを解決できるわけではありません**。

### 運営がすること ✅

1. **明確な利用規約違反への対処**
   - [モデレーション原則](/reference/moderation/)に明記された禁止事項への対処
   - 法令違反、システム攻撃、重大な脅迫など

2. **サーバーの技術的な管理**
   - サーバーの稼働維持
   - セキュリティ対策
   - 機能の追加・改善

3. **最終的な判断**
   - 重要な判断や異議申し立ての審査

### 運営がしないこと ❌

1. **ユーザー間の仲裁・調停**
   - 「AさんとBさんが揉めている」→ 運営は介入しません
   - ユーザー同士の問題は、ユーザー同士で解決してください

2. **グレーゾーンへの介入**
   - 「これは違反かどうか微妙...」→ 運営は介入しません
   - 疑義がある場合は、違反として扱いません

3. **予防的な監視・指導**
   - 「この人、問題を起こしそう」→ 運営は介入しません
   - 問題が起きる前に介入することはありません

4. **24時間365日の即座の対応**
   - 運営も人間です。すぐに対応できないこともあります

<Aside type="note" title="詳細は">
詳しい運営体制については「[運営体制と権限](/reference/administration/)」を参照してください。
</Aside>

## ユーザーに期待すること

やみすきーでは、**ユーザー自身が自律的に行動すること**を期待しています。

### 1. 自分を守る 🛡️

**ブロックやミュートを積極的に使ってください。**

- 今日は人の「やみ」を見たくない → やみモードOFF
- この人の投稿がつらい → ミュート
- この人と関わりたくない → ブロック

これは冷たいことでも、悪いことでもありません。**自分を守るための正当な手段**です。

<Aside type="tip" title="ブロック/ミュートは正当な権利">
[「見たくない」も正当な権利です](/guides/manner#見たくないも正当な権利です)も参照してください。
</Aside>

### 2. 自律的に行動する 🚶

**運営に依存せず、自分で判断してください。**

- ユーザー同士で解決できることは、解決する
- グレーゾーンは自分で判断する
- 運営に「どうすればいいですか？」と聞く前に、自分で考える

### 3. 明確な違反のみ通報する 📢

**通報する前に、本当に違反か確認してください。**

- [モデレーション原則](/reference/moderation/)の禁止事項に明確に該当するか？
- グレーゾーンではないか？
- ブロック/ミュートで解決できないか？

**グレーゾーンは通報しないでください。** 運営は対応しません。

## 運営に頼りすぎないで

やみすきーの運営も、みんなと同じように疲れることがあります。

だから、**運営に頼りすぎないでください**。

- 自分で距離を取れることは、自分で取る（ブロック/ミュート）
- ユーザー同士で解決できることは、解決する
- 運営に通報するのは、明確な違反だけ

運営は魔法使いじゃないから、すべてのトラブルを解決できるわけではありません。

**「運営に依存しない」ことが、やみすきーを守ることにつながります。**

## 通報する前に考えてほしいこと

通報する前に、以下を確認してください：

### チェックリスト

- [ ] [モデレーション原則](/reference/moderation/)の禁止事項に明確に該当するか？
- [ ] ブロック/ミュートで解決できないか？
- [ ] ユーザー間の問題ではないか？（ユーザー間の問題は運営は介入しません）
- [ ] グレーゾーンではないか？（グレーゾーンは運営は介入しません）
- [ ] 証拠（スクリーンショットなど）はあるか？

すべてにチェックが入った場合のみ、通報してください。

### 通報すべき例

- 「助けてくれないと死ぬ」のような脅迫的な要求
- 個人情報の無断公開
- 執拗なストーキング行為
- システム攻撃

### 通報すべきでない例

- 「あの人の投稿が不快」→ ミュートしてください
- 「AさんとBさんが揉めている」→ 運営は介入しません
- 「あの人、構ってちゃんだ」→ ブロックしてください
- 「これって違反かな...？」→ グレーゾーンは運営は介入しません

## 運営への感謝と理解

やみすきーの運営は、ボランティアです。

- 感謝の気持ちを伝える
- 問題があったら落ち着いて報告する
- 建設的な提案をする
- **運営も人間だと理解する**

運営に過度な期待をせず、自律的に行動することで、やみすきーは持続可能になります。

<Aside type="caution" title="重要">
運営への過度な依存は、やみすきーの存続を危うくします。

**ユーザー一人ひとりが自律的に行動することが、やみすきーを守ることにつながります。**
</Aside>
